Responsive

Bootstrap will figure out how wide your screen is and respond by resizing your HTML elements

  • hence the name Responsive Design.

img-responsive class可以让图片适应全屏。


Bootstrap uses a responsive grid system, which makes it easy to put elements into rows and specify each element’s relative width. Most of Bootstrap’s classes can be applied to a div element.
The col-md-* class is being used. Here, md means medium, and * is a number specifying how many columns wide the element should be. The column width of an element on a medium-sized screen, such as a laptop, is being specified.

col-xs-*, where xs means extra small (like an extra-small mobile phone screen)
One row has 12 grid.






Bootstrap has a class called well that can create a visual sense of depth for your columns.
